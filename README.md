# Important

##### dot-transform is a function that solves the price problem



## HOW TO USE

dot-transform receives a number and transforms it.

 
- 1000

###### Results
- 1,000


## Installation
 
```sh
npm i dot-transform
```
```sh
use
import dotTransform from dot-transform

dotTransform(10000), returns 10,000
```
